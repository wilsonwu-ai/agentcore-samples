ℹ️ About this deployment

This is an operator-deployed demo content provider for the Pay for Content (Browser Use) sample. The paywall uses the x402 protocol — payment requirements are embedded in the page DOM so an AI browser agent can read them without parsing HTTP headers.

This content provider uses CloudFront + Lambda@Edge for x402 paywall verification. It is intended as a test target for the sample notebook only.
